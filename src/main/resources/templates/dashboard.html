Удаление аккаунта

Что произойдет при удалении аккаунта:

  • Ваш аккаунт будет полностью удален из системы
  • Все ваши личные данные будут удалены без возможности восстановления
  • История звонков будет анонимизирована - ваши данные не будут видны другим пользователям
  • Вы потеряете доступ ко всем функциям системы
  • Это действие нельзя отменить

Ваши данные не будут видны другим пользователям после удаления. История звонков будет сохранена в анонимизированном виде для статистики.
